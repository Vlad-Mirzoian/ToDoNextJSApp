- Next.js 14

## Startup

```bash
git clone https://github.com/Vlad-Mirzoian/ToDoNextJSApp.git
cd ToDoNextJSApp
npm install
cp .env.example .env
# In .env, insert your MongoDB string:
# MONGODB_URI=mongodb+srv://...
npm run dev
```
Open: http://localhost:3000
